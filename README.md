# chudbase, a bare bones internal for cs2

## what is this
chudbase is an internal base for beginners (or just someone who doesn't want to start a base from scratch) cs2 which includes almost nothing more than what is needed. it's just to get you started without having to do the boring parts of making a base

## features
- interface system
- schema system
- convar system
- full hooking system using minhook
- pre made dx11 hook with gui
- pre made OnAddEntity and OnRemoveEntity hooks
- pre made OverrideView hook (with CViewSetup)
- pre made OnRenderStart hook
- xrefs for every pattern

## how to use
open the project in visual studio and build, inject with your favorite manual map injector.
